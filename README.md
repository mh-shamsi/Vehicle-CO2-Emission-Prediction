# 🚗 Vehicle CO2 Emission Prediction

A machine learning project for predicting vehicle CO₂ emissions based on engine and fuel consumption features.

## 📌 Overview

This project explores the relationship between vehicle specifications and CO₂ emissions using regression models. It includes data preprocessing), model training, and performance evaluation.

## 📊 Dataset

The dataset contains vehicle characteristics such as:

- Engine Size
- Cylinders
- Fuel Consumption (City)
- Fuel Consumption (Highway)
- Fuel Consumption (Combined)
- Fuel Type
- Transmission
- Vehicle Class

**Target:**

- CO₂ Emissions (g/km)

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook


## 🤖 Machine Learning Workflow

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Training
- Model Evaluation

## 📈 Model

This project uses regression algorithms to estimate vehicle CO₂ emissions.

Evaluation metrics include:

- R² Score
- Mean Squared Error (MSE)
